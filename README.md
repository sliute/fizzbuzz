# Fizzbuzz

Fizzbuzz

A [Makers Academy precourse challenge](https://github.com/makersacademy/pre_course/blob/master/exercises/fizzbuzz.md).

## How to Use
```
$ git clone https://github.com/sliute/fizzbuzz.git
$ cd fizzbuzz/lib
$ irb
$ require './fizzbuzz'
$ 1.upto(100) { |x| puts fizzbuzz(x) }
```
